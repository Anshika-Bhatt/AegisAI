# Aegis AI - Industrial Safety Intelligence Platform

## Problem Statement

Industrial accidents often occur because critical warning signals are not interpreted quickly enough. Operators receive raw alerts but lack contextual intelligence for rapid decision-making.

## Solution

Aegis AI combines machine learning and AI-powered safety recommendations to identify high-risk industrial situations and provide actionable guidance.

## Features

* Industrial Risk Prediction using Random Forest
* Synthetic Industrial Sensor Dataset
* Safety Knowledge Base
* AI Safety Copilot
* Streamlit Dashboard
* Industrial Safety Architecture Visualization

## Tech Stack

* Python
* Streamlit
* Scikit-Learn
* Pandas
* ChromaDB
* Sentence Transformers
* Gemini API (planned integration)

## Project Workflow

Sensor Data
→ Risk Prediction Engine
→ Safety Assessment

Safety Documents
→ Embeddings
→ Vector Database
→ Safety Copilot
